巴蜀文化资源汇编

千年积淀，文化瑰宝——全方位展现巴蜀文化的深度与广度

一、非遗技艺：匠心传承千年

巴蜀地区是非遗资源富集地，拥有国家级非遗项目150余项，省级非遗项目500余项，涵盖传统技艺、传统戏剧、传统美术等多个类别。

1. 蜀绣：中国四大名绣之一，以针法细腻、色彩艳丽、图案精美著称，代表作《芙蓉鲤鱼》《熊猫》等，远销海内外。

2. 蜀锦：始于战国，兴于汉唐，以"经纬提花、色彩斑斓"闻名，是中国古代丝织工艺的巅峰之作，现为国家级非遗。

3. 川剧：融合唱、念、做、打，以变脸、吐火、滚灯等绝技为特色，是巴蜀文化的活化石，2006年被列入国家级非遗。

4. 自贡灯会：始于唐宋，盛于明清，以"形、色、声、光、动"为特色，被誉为"天下第一灯"，每年吸引数百万游客。

二、民俗文化：多元风情交融

巴蜀地区民族众多（汉族、彝族、藏族、羌族等），形成了丰富多彩的民俗文化，展现了"天府之国"的包容与活力。

1. 春节民俗：贴春联、挂灯笼、吃团年饭、逛庙会（成都武侯祠庙会、重庆磁器口庙会），热闹非凡。

2. 端午习俗：四川新津赛龙舟、重庆合川龙舟赛，传承千年龙舟文化；吃粽子、挂艾草、佩香囊，祈福避灾。

3. 彝族火把节：凉山彝族自治州的传统节日，每年农历六月二十四，人们点燃火把、围着篝火跳舞，祈求丰收与平安。

4. 羌族瓦尔俄足节：羌族女性的传统节日，又称"妇女节"，以祭祀、歌舞、刺绣展示为主要内容，体现羌族女性文化。

三、历史底蕴：文明源远流长

巴蜀地区是中华文明的重要发源地之一，从三星堆文明到金沙遗址，从都江堰到武侯祠，每一处遗址都见证着千年历史变迁。

1. 三星堆文明：距今3000-5000年，出土青铜神树、纵目面具、黄金面具等珍贵文物，展现了古蜀文明的神秘与辉煌。

2. 金沙遗址：古蜀王国的都城遗址，出土太阳神鸟金箔、青铜神树等文物，太阳神鸟已成为中国文化遗产标志。

3. 都江堰水利工程：战国时期李冰父子修建，至今仍在发挥防洪、灌溉作用，是世界上现存最古老的水利工程之一，被列入世界文化遗产。

4. 武侯祠：纪念三国时期诸葛亮的祠堂，位于成都，是中国唯一的君臣合祀祠庙，体现了巴蜀人民对诸葛亮的敬仰。

四、艺术文化：雅俗共赏

巴蜀艺术融合了宫廷艺术与民间艺术，形成了"雅俗共赏、刚柔并济"的独特风格。

1. 川菜文化：中国八大菜系之一，以麻辣鲜香为特色，分为蓉派（成都）、渝派（重庆）、盐帮菜（自贡）等，火锅、回锅肉、麻婆豆腐等菜品享誉世界。

2. 巴蜀文学：古代有司马相如、李白、苏轼等文学巨匠，现代有巴金、郭沫若等文坛大师，形成了独特的"巴蜀文学流派"。

3. 川派盆景：中国五大盆景流派之一，以"虬曲多姿、苍劲古朴"为特色，融合了自然美与艺术美。

4. 巴蜀音乐：川剧音乐、四川清音、羌族多声部民歌等，展现了巴蜀地区丰富的音乐文化。
